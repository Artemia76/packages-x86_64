Produce a customised ISO

All the commands should be enter in the root account

  1. Installation of a NuTyX

    1. Get all the tools

      git clone git://github.com/NuTyX/houaphan.git
wget http://downloads.nutyx.org/enter-chroot{,.md5sum}
wget http://downloads.nutyx.org/install-houaphan{,.md5sum}
md5sum -c enter-chroot.md5sum
      enter-chroot: OK
      md5sum -c install-houaphan.md5sum
      install-houaphan: OK

    2. Make shure you have cdrkit and syslinux installed, in worth case:

      get cdrkit
get syslinux

    3. A variable will be used during the all process

      The LFS variable MUST be defined

      export LFS=/ISO-MINI

      The chroot will be in the "/ISO-MINI" directory. Up to you to choose another place.

    4. Install the houaphan base system

      bash install-houaphan

    5. Copy the files for the iso

      cp -av houaphan/iso $LFS/ISO

    6. Enter the houaphan base system

      bash enter-chroot

    7. Optionnal: if you have everything on your local host

      sed -i "s@downloads.nutyx.org@localhost@" etc/cards.conf

    8. Get the meta datas

      cards sync

    9. Needed tools for the installer

      Depending of your file system choice, install:

      For ext2, 3 et 4:

      get e2fsprogs

      For reiserfs:

      get reiserfsprogs

      For xfs:

      get xfsprogs

      For jfs:

      get jfsutils

      For btrfs:

      get btrfs-progs

      Password handler

      get sudo

      Gestionnaire des inits et logs

      get sysklogd
get sysvinit

      DHCP Service

      get dhcpcd

      Grub the boot system

      get grub

      Localisation and communication data

      get tzdata
get iproute2
get iana-etc

      Text editor

      get e3

      or

      get vim

      Utilities for the kernel and init of the machine

      get eudev
get kmod

      Documentation

      get less
get groff
get man-db
get cards.man
get nutyx.man

      Utilities for the iso:

      get lvm2
get mdadm
get dialog
get cpio
get squashfs

      Firmwares

      get linux-firmware

      A kernel

      get kernel

      or

      get kernel-lts

      or

      get kernel-iso

  2. Generate the ISO

    1. Put the kernel and the initrd in the right place

      cp -v /boot/kernel-* /ISO/isolinux/kernel
cp -v /boot/initrd-* /ISO/isolinux/initrd

    2. Optionnal: Reset the default adress of the NuTyX mirror

      sed -i "s@localhost@downloads.nutyx.org@" etc/cards.conf

    3. Flush the binaries archives

      cards purge

    4. Generate the squashfs files 

      mkdir -p /ISO/isolinux/boot/
cd /
for dir in opt bin etc lib root run sbin usr var home
do
  [ -f ISO/isolinux/boot/$dir.squashfs ] && rm ISO/isolinux/boot/$dir.squashfs
  mksquashfs $dir ISO/isolinux/boot/$dir.squashfs
done

    5. Quit the NuTyX chroot

      exit

    6. Generate the ISO

      sh houaphan/scripts/mkiso houaphan-`date +%Y%m%d`

  3. Conclusion

    The size shoul'nd be bigger then 300 MB if you didn't put anything else then all the packages mentionned above. You will find the iso in the $LFS directory.

    ls $LFS
    bin   dev  home  lib  NuTyX_x86_64-houaphan-20150723.iso     proc  run   srv  tmp  var
boot  etc  ISO   mnt  NuTyX_x86_64-houaphan-20150723.md5sum  root  sbin  sys  usr
