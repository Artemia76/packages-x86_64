Installation

Make sure you have enough space on your hard disk. 8 GiB to 15 GiB should be enough.

Download the latest version of NuTyX

The current release of NuTyX is named houaphan 8.1. The available ISO image can be used to install all the packages from the Internet. It is also used to carry out a minimum of configuration of your machine before the actual installation of NuTyX.

Both isos are +- 210 Mb.

All the available ISO images can be downloaded at: http://downloads.nutyx.org

Create a bootable USB key

If you are currently running GNU/linux on your machine:

As root

With the help of fdisk, find out where your usb key is located:

fdisk -l

Then by using the command dd, create a bootable key:

dd if=NuTyX_x86_64-houaphan-8.1.iso of=/dev/sdX

The sdX value is the one that you have previously found using fdisk.

If you are using Windows, it is easier to burn the ISO image onto a CD.

Typical installation of NuTyX on a single partition

How to install NuTyX from scratch on your PC

The default root password is nutyx. Once you have successfully logged in, you will be required to change the password.

Installing a bootloader

How to install and configure the bootloader for your PC

If you used the latest grub version 2.XX , the following entry can be added to your grub.cfg:

menuentry "NuTyX sur /dev/sda1" {
    set root=(hd0,1)
    linux /boot/kernel root=/dev/sda1 ro quiet
    initrd /boot/initrd
    }

    menuentry "Windows 8" {
    set root=(hd0,3)
    chainloader (hd0,3)+1
    }
