NuTyX saravane 15.05 available with cards 1.0

Introduction for the impatient

Seven months after the release of saravane 14.11 instead of the three months originally planned, I am very proud to annonce you (finally) the release of NuTyX saravane 15.05 which in summary the new features:

  • glibc 2.21
  • gcc 4.9.2
  • binutils 2.25
  • kernel 3.19.6
  • kernel-lts 3.18.12
  • kernel-iso 3.18.11
  • cards 1.0
  • curl 7.42.1
  • firefox 37.0.2
  • Convenient configuration tool from the command line for the clock, keyboard, network and language: setup-nutyx
  • Ready to use User Graphic Interface for installation in any Simplicity (with auto login): KDE 4.14.3 and 4.10.1 Xfce4
  • New iso 15.05 in 32 and 64 bits
  • New completly revised version of cards finaly available in 1.0
  • Documentation completed on the web site
  • Installation script of NuTyX saravane
  • The complete project is available at https://github.com/NuTyX

For full update packages, you can visit:https://github.com/NuTyX/saravane/commits/master

The iso

If the work of partitioning and formatting of the destination partition was made upstream and if the destination partition is unique, the installation is done very quickly since you only have to choose: "install" in the main menu.

The installation of the boot process has also been simplified: If you install GRUB BEFORE NuTyX, you can choose the destination partition. If you install NuTyX IMMEDIATELY and choose (at the end of the installation of NuTyX) to GRUB, it will be AUTOMATICALY installed on the partition where NuTyX is installed.

The kernel used is a 3.19.6. An LTS release (Long term support) is also available in Version 3.18.12.

The iso is now translated to 100% in the following languages: French, Dutch, Finnish, Swedish, Norwegian and German.

The file system jfs has been added as possible format. It seems that this is not the best file system we can find today. Performance level, there is the btrfs, stability and control level there is xfs.

If you want to install on a formatted NuTyX btrfs partition, you must install grub on a separate and especially not install grub partition (again) on the partition of NuTyX. For example /dev/sda1 formatted in ext2 will be set to boot from the machine and /dev/sda2 formatted in btrfs will be used for install NuTyX.

The installation script

It is now possible to install very easily NuTyX from any distribution. The latest problems with the version of the BASH interpreter have been resolved.

Two variables are good to knows:

LFS
URL

The LFS variable give the possibility to choose a destination folder. Ones define and "exported", the installation command get as simple as:

bash install-saravane

The URL allows you to define a personnal mirror. Ones the installation finish, cards will still used this mirror.

The cards package manager is finaly release in 1.0 version

In its final version, cards is now even faster and easier to use. The most common commands have been simplified:

To synchronise information with the default mirror:

cards sync

To know the installed packages:

cards list

For details of an installed package:

cards info <name_of_my_package>

To check the obsolete list of installed packages:

cards diff

The command:

man cards

will give you all the details about the available commands, some of which are new:

cards query
cards files
cards help
cards purge
cards depcreate

As for the packager, new variables have been added:

PKGMK_LOCALES

Default value:

PKGMK_LOCALES=(fr)

Define the locales that we want to keep in the binary, example: PKGMK_LOCALES=(fr nl) will conserve the french and italians locales (if they were available at compile time) in the sub packages <name_package>.fr and <name_package>.it

PKGMK_IGNORE_REPO

Default value:

PKGMK_IGNORE_REPO="yes"

If set as "no", the file .PKGREPO of the collection (means the name of the parent folder of your package) will be (re)generate. This file is needed if you wish to installed some packages outside of your choot
