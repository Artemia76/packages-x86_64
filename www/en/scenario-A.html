Scenario A

Introduction

You are not wanting to compile anything on your NuTyX. However, if you are wanting other specific packages that are not included in the below “binary-collection”, then you need to reconsider your options . Works perfect if you want to do multiple NuTyX installations, and prefer the system to automatically do the installation on your behalf.

The /etc/cards.conf file

All the below binary collection are activated.

The architecture chosen for your machine is correct.

The chosen language(s) for the machine have been selected.

The folder defining the base system was created.

The contens of a typical /etc/cards.conf file:

dir /var/lib/pkg/depot/xfce4-extra|http://downloads.nutyx.org
dir /var/lib/pkg/depot/xfce4|http://downloads.nutyx.org
dir /var/lib/pkg/depot/gui-extra|http://downloads.nutyx.org
dir /var/lib/pkg/depot/gui|http://downloads.nutyx.org
dir /var/lib/pkg/depot/cli-extra|http://downloads.nutyx.org
dir /var/lib/pkg/depot/cli|http://downloads.nutyx.org
dir /var/lib/pkg/depot/base|http://downloads.nutyx.org
arch x86_64
locale de
base /var/lib/pkg/depot/base
base /var/lib/pkg/depot/base-extra

The 6 standards binaries collections base, cli and gui are activated, as well the xfce4 and xfce4-extra.

The processor is a 64 bits x86_64.

The language choose for the installation is german locale de, for english you can ignore this line as per default it will be in english.

A cards base -r command is used to return to a base system. The folder containing the base packages is the one of the standard base collection.

base /var/lib/pkg/depot/base
base /var/lib/pkg/depot/base-extra

The /etc/pkgmk.conf file

In scenario A, the /etc/pkgmk.conf file is present but since you do not compile, it will not be used. Thus the development tools are not needed.

Conclusion

That's all you need to set up your NuTyX. To update your database and check for any available updates, use command:

check

Want to install a Internet browser package, use command:

get firefox

To delete a package:

del firefox

That's all there is to know . I hope you'll get plenty of enjoyment using NuTyX.
