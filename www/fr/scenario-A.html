Scénario A

Introduction

Vous ne souhaitez pas compiler dans votre NuTyX. Ce qui est tout à fait possible si vous n'avez pas besoin de programmes particuliers et que les binaires proposés vous suffisent. Ce scenario est aussi valable dans le cas où vous gérer plusieurs installations sur les quelles aucune compilation n'est necessaire.

Le fichier /etc/cards.conf

Les collections binaires sont activées

La(les) langue(s) utilisée(s) sur la machine est ajustée(s)

Les dossiers définissant le système de base, en cas de retour à un système de base sont définis

Le contenu typique de votre fichier /etc/cards.conf:

dir /var/lib/pkg/depot/xfce4-extra|http://downloads.nutyx.org
dir /var/lib/pkg/depot/xfce4|http://downloads.nutyx.org
dir /var/lib/pkg/depot/gui|http://downloads.nutyx.org
dir /var/lib/pkg/depot/gui-extra|http://downloads.nutyx.org
dir /var/lib/pkg/depot/cli|http://downloads.nutyx.org
dir /var/lib/pkg/depot/cli-extra|http://downloads.nutyx.org
dir /var/lib/pkg/depot/base|http://downloads.nutyx.org
dir /var/lib/pkg/depot/base-extra|http://downloads.nutyx.org
locale fr
locale de
base /var/lib/pkg/depot/base
base /var/lib/pkg/depot/base-extra

Les six collections standards base, cli et gui sont activées, ainsi que les collections xfce4. Les langues qui doivent être installées sont le français fr et l'allemand de. Les dossiers contenant les paquets de base snt ceux des collections standard base et base-extra.

base /var/lib/pkg/depotrepo/base
base /var/lib/pkg/depotrepo/base-extra

Le fichier /etc/pkgmk.conf

Pour le scénario A, même si le fichier /etc/pkgmk.conf est présent, il ne sera pas utilisé puisque vous ne devez rien compiler. Les outils de développement ne sont pas présents et pas nécessaires.

Conclusion

C'est tout ce qu'il faut savoir pour la configuration de votre NuTyX. Vous souhaitez mettre à jour votre base de données et voir si il y a des maj disponibles

check

Vous souhaitez installer un paquet:

get firefox

Vous souhaitez supprimer un paquet:

del firefox

C'est tout ce qu'il faut savoir . J'espère que vous allez vous en sortir.
